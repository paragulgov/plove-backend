Приложение для прогнозов на футбольные матчи. Можно играть с друзьями и смотреть, кто больше угадывает.

[Demo](https://plove.herokuapp.com). Есть админка, но она не доступна в демке.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
